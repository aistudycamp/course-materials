Pair Programming Challenge

Build a Collaborative Board.

Scenario: Figma & Miro crashed, but you still need to lead a team brainstorm in 1 hour. Ship a real-time collaborative board — complete with authentication, live cursors, and shared rooms.

What you'll build

Five features. One board.

01
Users log in via Magic Link (authentication).
02
Once logged in, there's one multiplayer board ('room').
03
Users can add rectangles, sticky notes, to-dos, and text to the board.
04
Any authenticated user can create, read, update, and move items ('permissions').
05
Users can see who else is viewing the board ('presence' and 'cursors').
multi-player presence cursors 'live' rooms authentication
Pro tips

Three pro tips.

Set yourself up so the first thirty minutes don't get eaten by setup.

01Scaffold first
Once your project is created, run this from your VS Code terminal — before calling Claude. It creates an Instant + Next.js app for you:
npx create-instant-app
02See it live
Open a second terminal window and run your local environment without blocking Claude:
npm run dev
Make sure you're in your 'smallest' project folder.
03Read the docs
Check out Instant's docs — especially Presence, Rooms, and Cursors:
instantdb.com/docs/presence-and-topics
