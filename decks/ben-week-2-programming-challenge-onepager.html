Week 2 · Pair Programming Challenge

Build a Shout-Out Wall.

Scenario. You really like your classmates at AI Study Camp — and you want to share the love. Build a real-time wall where anyone can post a quick message of appreciation for a classmate.

What you'll build
01
Input the name of the person posting.
02
Input who's getting the appreciation + a short message.
03
A real-time feed showing all shout-outs.
04
A "👏" counter so others can clap for a post.
05
Sort feed by most clapped (highest → lowest).
Bonus
📷 Image posts
Add an image alongside the text.
🎉 Confetti on clap
Trigger confetti when someone claps.
Concepts
real-time updates database integration list rendering optimistic UI

If you finished Sprint 1, you have the tech stack and know how to do this.
