Which one, when.

Three products, three distinct jobs. Picking the wrong one costs time.

01
Claude Chat
Reach for this when
  • You need a quick answer, draft, or research — right now, on your phone
  • You're brainstorming or iterating on writing and want back-and-forth
  • You want web search without any setup or configuration
Skip it when
  • You need to work with files on your computer
  • You want the same task to repeat automatically
02
Claude Cowork
Reach for this when
  • You need a real Office document — Word report, Excel tracker, PowerPoint deck
  • You want a recurring workflow (daily briefing, weekly summary) with no coding
  • You're handing a system to a non-technical teammate — ZIP the folder, they open it and go
Skip it when
  • You need to run, test, or ship code
  • You need scheduled tasks that run when your laptop is closed
  • You need version control — outputs save locally only, no git, no backup
  • Your team needs to stay in sync — sharing is manual ZIP or Drive, not a live repo
03
Claude Code
Reach for this when
  • You're writing, debugging, or shipping software — this is what it was built for
  • You need full git — commits, PRs, push to GitHub, everyone on the team always has the latest
  • You need automation that runs in the cloud on a schedule, even when your laptop is off
Skip it when
  • You need native Office document output
  • You're building something for a teammate who doesn't use a terminal — use Cowork instead
