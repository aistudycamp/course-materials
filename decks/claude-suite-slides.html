01
Claude Chat
claude.ai Desktop · Mobile
Good at
Q&A, writing, research, brainstorming, web search, document analysis
Can't do
Read or write files on your computer, run commands, autonomous multi-step tasks
Worth knowing
Connects to Gmail, Calendar, Notion, Slack via integrations. Only product with a free plan.
02
Claude Cowork
Desktop app only
Good at
Native Office docs with real formulas, M365 + SharePoint integration, browser automation, scheduled recurring tasks, 11 role plugins (Sales, Legal, Finance…)
Can't do
Git, package managers, full filesystem access, MCP servers, custom skills or hooks
Worth knowing
Computer must stay awake; tasks pause if machine sleeps. Eats message quota fast. Desktop app only. Still beta (Apr 2026).
03
Claude Code
CLI Desktop Web VS Code
Good at
Full codebase + git, terminal, deploy, MCP servers, skills, sub-agents, Routines — cloud-scheduled agents triggered by time, API call, or GitHub event
Can't do
Native Office documents. Screen control limited in CLI. No visual diffs without Desktop app.
Worth knowing
Context fills fast — clear at 40–50%. Routines run in the cloud; laptop can sleep.

All three products run the same underlying Claude model. What differs is the permission model, interface, and what actions Claude is allowed to take on your behalf.
