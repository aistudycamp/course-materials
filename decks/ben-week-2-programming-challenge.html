Week 2 · Pair Programming Challenge

You really like your classmates at AI Study Camp — and you want to share the love.

Build a real-time "Shout-Out Wall" where anyone can post a quick message of appreciation for a classmate.

What you'll build

Five features. One feed.

01
Input the name of the person posting.
02
Input who's getting the appreciation + a short message.
03
A real-time feed showing all shout-outs.
04
A "👏" counter so others can clap for a post.
05
Sort feed by most clapped (highest → lowest).
📷 Bonus · image posts
Drop an image alongside the text.
🎉 Bonus · confetti on clap
Trigger confetti when someone claps.
real-time updates database integration list rendering optimistic UI
Pro tips

Three pro tips.

Set yourself up so the first thirty minutes don't get eaten by setup.

01Scaffold first
From your VS Code terminal — inside your GitHub project, before calling Claude — run this to spin up an Instant + Next.js app:
npx create-instant-app
02Use your tools
Don't forget your CLAUDE.md file. And use SHIFT + Tab to drop into Plan Mode before you ship code.
Plan Mode = think before you let it write.
03See it live
Open a second terminal window in VS Code so you can run your local environment without blocking Claude:
npm run dev
