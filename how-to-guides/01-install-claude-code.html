Guide 01

How to Install and Set Up Claude Code

Get Claude Code running on your machine in under 10 minutes.

Prerequisites

  • macOS, Windows, or Linux
  • Node.js v18+ — download the LTS from nodejs.org
  • An Anthropic account — console.anthropic.com
1

Install Claude Code

With Node installed, run:

zsh
$ npm install -g @anthropic-ai/claude-code
Tip

Permissions error? Prefix with sudo.

2

Start Claude Code

Navigate to any folder and run claude. The first run opens your browser to authenticate with Anthropic — follow the prompts and return to your terminal.

zsh
$ cd ~/Desktop $ claude
3

Verify it works

At the Claude Code prompt, ask:

claude
What files are in this directory?

If Claude responds with a list of files, you're all set.

Troubleshooting

"command not found: claude"

Open a fresh terminal window so the new command registers.

"npm not found"

Reinstall Node.js LTS from nodejs.org.
